Ordering

The current API implements ordering calls on the ActivityResource date fields.

{{ api_url }}/activities/?format=json&order_by=start_planned
Note: ordering can be reversed by adding a minus sign before the field. Also order_by can be joined with other filters.

Fields

The current ActivityResource implements (reverse) ordering on: start_planned, start_actual, end_planned, end_actual
